## 🌙 v0.26.0 — LevelUp 카드 픽셀아트 + Galmuri 폰트 + 다국어 layout 안전화

### 🎯 핵심 변경

레벨업 카드에 픽셀아트 자산(panel_card_dark + rarity 글로우 4색)을 통합하고, 시스템 폰트(Pretendard)를 픽셀아트 한글 폰트(**Galmuri 11**)로 일괄 교체. 메뉴 layout을 디자인 좌표 기준으로 가운데 정렬하고, 영문/한국어 둘 다 안전하게 fit하도록 폭/폰트 사이즈를 재조정.

---

### 🎨 LevelUp 카드 픽셀아트 통합 (Phase UI-4)

- **카드 panel** `panel_card_dark.9.png` (128×160) — StyleBoxTexture로 적용, 어두운 stone tablet 톤
- **rarity 글로우 overlay** — NinePatchRect 동적 추가
  - NEW (신규 무기) → 파랑
  - UP (강화) → 호박
  - EVOLVE (진화) → 보라
  - PASSIVE NEW → 초록
  - PASSIVE UP → 호박
- **글로우 자산 가공** — 채도 기반 알파 (saturation < 40 → alpha 0). 무채색(흰/회) 모두 알파 처리, 진한 색만 보존
- **NinePatchRect.draw_center = false** — 가운데 stretch 영역 안 그림, 외곽 글로우만
- **glow.self_modulate = WHITE** 강제 — PanelContainer modulate가 글로우 색 변형하지 않도록
- **SELECT 버튼 외곽선** — 진한 navy outline 6px, 어떤 rarity 색 배경에서도 가독성 보장
- **폰트 사이즈** — 카드 title 42, header tag 30, stats/desc 28~30, select 32
- **카드 높이** 372 → 420, 아이콘 96 → 108

### 📐 메인 메뉴 다국어 layout 안전화

- **VBox 가운데 정렬 + 폭 720** — `anchor_left/right=0.5`, offset −360~360. 화면 폭 변화에 관계없이 메뉴 폭 일관
- **`stretch_aspect=expand`** 복구 + `default_clear_color` 어두운 navy로 — letterbox 영역도 디자인 톤 통일
- **size_flags_horizontal=3 + clip_text 일괄 명시** — BtnPlay, StatusCard, PrimaryRow/SecondaryRow/TertiaryRow 모두. 영문 길이로 인한 의도치 않은 sizing 차이 제거
- **폰트 사이즈 영문 fit 보정**
  - BtnPlay 76 → 60 (영문 PLAY 폭이 한글 시작보다 커서)
  - StatusLabel 30 → 24 (영문 "Pyromancer · Forest of Echoes · Normal" 길이)
  - NextGoalLabel 28 → 22 (영문 "★ Upgrade ready" 길이)
- **영문 텍스트 단축** — `menu_next_goal_fmt` "Next upgrade in %d gold" → "%d g to upgrade", `menu_next_goal_ready` "★ Upgrade ready · open Shop" → "★ Upgrade ready", `menu_next_goal_maxed` "All upgrades maxed" → "All maxed"
