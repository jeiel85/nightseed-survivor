## 🌙 v0.30.0 — 스테이지 첫 클리어 자동 해금

### 🎯 핵심 변경

뱀파이어 서바이버 계열 장르 관습에 맞춰, 스테이지 **첫 클리어 시 다음 스테이지가 골드 차감 없이 자동으로 해금**됩니다. 더 이상 스테이지 해금 = 골드 차감이 아니라, 일단 정공법 진행이 가능하고 골드 직접 해금은 빠른 패스로 남아 있습니다. 추가로 Hard / Nightmare 난이도 첫 클리어 시 일회성 보너스 골드, Cursed Tomb (마지막 스테이지) 첫 클리어 시 엔딩 메시지가 결과창에 표시됩니다.

---

### 🆕 추가

- **스테이지 자동 해금** — 어떤 난이도든 처음으로 스테이지를 클리어하면 `stages.json` 의 `next_stage` 필드가 가리키는 다음 스테이지를 즉시 영구 해금. 결과창에 `★ 다음 스테이지 해금: …` 라인이 표시됩니다.
- **난이도 첫 클리어 골드 보너스** — 같은 (스테이지, 난이도) 쌍을 처음 깬 경우 일회성 골드 보너스. Hard +500, Nightmare +1000. Normal 은 해금만 (보너스 없음).
- **캠페인 완주 메시지** — Cursed Tomb 첫 클리어 시 `★ 모든 스테이지 정복 — 영원한 밤이 잠들었습니다.` 라인 추가.
- **StageSelect 카드 클리어 표시** — 카드마다 `클리어:  ★ Normal  ★ Hard` 형태로 클리어한 난이도가 노란색 라인으로 노출.

---

### 🛠 구현 메모

- `GameData.stages_cleared` 영구 기록 — `{stage_id: ["normal", "hard", ...]}` 형태. PGS Snapshots 머지 정책은 union, 저장은 다른 진행도와 함께 즉시 flush.
- 자동 해금과 보너스 골드는 둘 다 `GameRoot._on_victory()` 시점에 적용되고 `GameData` 에 즉시 저장 — 결과창에서 앱을 그냥 꺼도 보상이 유지됩니다.
- 광고 Double Gold 와 보너스 골드는 완전히 독립 (광고는 in-run 골드만 2배, 보너스는 별도 가산).
- 새 Localization 키 4개: `result_stage_unlocked_fmt`, `result_first_clear_bonus_fmt`, `result_campaign_finished`, `stage_cleared_fmt` (en/ko).
