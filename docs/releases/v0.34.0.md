## 🌙 v0.34.0 — 스토리 화면 폴리시: 잠금 안내·스포일러 가림·점프 탭

### 🎯 핵심 변경

v0.33.0에서 새로 연결한 Story Chronicle 상세 장부를 실제 폰 사용 흐름에 맞춰 다듬었습니다. 잠긴 카드의 안내가 더 친절해지고, 클리어하지 않은 챕터 섹션의 결말 스포일러를 가리며, 본문 가독성이 한 단계 올라갑니다. 또한 5개 스테이지 인장을 상단 가로 탭으로 노출해 특정 챕터로 곧장 점프할 수 있게 했습니다.

---

### 🛠 변경 — 잠금 안내·스포일러·가독성

- 잠긴 스테이지 카드의 안내 문구를 일반적인 메시지 대신 **이전 스테이지 이름을 직접 가리키는** 형태로 교체했습니다.
  - 예: 얼어붙은 황무지 카드 → `「메아리의 숲」 클리어 후 해금됩니다.`
- **잠긴 챕터 섹션의 제목**을 `???` 로 마스킹했습니다.
  - 기존엔 클리어 전 사용자에게도 `남은 조각` 같은 결말 키워드가 노출되던 문제.
  - 본문 잠금 안내 문구는 기존과 동일하게 유지.
- **챕터 섹션 본문 폰트**를 20 → 22, 제목을 22 → 23 으로 상향. 한국어 장문(섹션당 350~400자)을 폰 720×1280 환경에서 읽기 편하도록 키웠습니다.

---

### 🆕 추가 — 상단 스테이지 인장 가로 탭

- 스토리 화면 헤더와 스크롤 영역 사이에 **5개 스테이지 인장**을 가로로 배치한 빠른 탭을 추가했습니다.
- 인장을 누르면 해당 스테이지 카드 위치로 자동 스크롤됩니다.
- **잠긴 스테이지의 인장도 클릭 가능** — 어떤 챕터가 남았는지 빠르게 확인할 수 있도록 잠금 카드로 스크롤됩니다 (디밍 표시).

---

### ✅ 검증

- `Localization.gd`에 `story_locked_prev_stage_fmt` 신규 키 추가 (ko/en) — 다른 호출부 영향 없음.
