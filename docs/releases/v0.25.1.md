## 🌙 v0.25.1 — 한국어 게임 제목 정리 + 메타 데이터 보정

### 🎯 핵심 변경

v0.25.0의 픽셀아트 타이틀 로고가 "잔불의 밤"으로 바뀌었지만 안드로이드 앱 이름, 브랜딩 페이지, 메타 데이터는 옛 "밤의 씨앗: 서바이버" 표기가 남아 있었음. v0.25.1은 한국어 표기를 **"잔불의 밤"**으로 일괄 정리하고, 영문은 **"Nightseed Survivor"**를 유지. 게임 로직 / 자산 변화 없음.

---

### 🇰🇷 한국어 게임 제목 일괄 변경

**`godot/project.godot` — 언어별 앱 이름**
- `application/config/name` "밤의 씨앗: 서바이버" → `"Nightseed Survivor"` (default, 영문)
- `application/config/name_localized` 추가: `{"en": "Nightseed Survivor", "ko": "잔불의 밤"}`
- 한국어 폰 → 앱 아이콘 아래 "잔불의 밤" 표시
- 영어/기타 언어 폰 → "Nightseed Survivor" 표시

**`branding/index.html` — GitHub Pages 메인 페이지** (10곳 일괄)
- `<title>`, `<h1>`, `og:title`, `og:site_name`, `twitter:title`, `og:image:alt` 등
- 검색어 keywords에 "잔불의 밤" 추가 + 기존 "밤의 씨앗" 보존 (검색 유입 호환)
- "7분 생존 액션" 잔재 → "5분 생존 액션" (v0.18부터 5분 정책에 맞춰 정정)

**보존된 표기 (의도된 그대로)**
- 게임 lore 용어 "밤의 씨앗" — `docs/story/`의 Nightseed 한국어 번역. lore 개념이라 보존
- 영문 코드네임 "Nightseed Survivor" — 프로젝트 이름, 외부 시스템 등록명 (Play Console, PGS, AdMob)
- 과거 release 노트의 옛 표기 — 역사 기록

---

### ⚠️ Known Issue (계속 — 다음 release에서 fix)

**★ RANK 버튼 클릭 시 PGS 리더보드가 열리지 않음** + AdMob 광고 표시 안 됨

- 원인: Godot 4.2 헤드리스 export(`--export-release`)가 EditorPlugin(AndroidExportPlugin)을 활성화하지 못해 PGS / AdMob native (.aar)이 빌드에 포함 안 됨
- 검증 시도: 헤드리스 `--editor 1800 frame` 사전 띄움 + `-v` verbose 빌드도 효과 없음. 자세한 진단은 v0.25.0 노트 참고
- 해결책 — **Godot GUI 에디터에서 직접 export** (Project → Export → Android/AAB → Export Project)
- 다음 release(v0.25.2 또는 v0.26.0)는 GUI 에디터 빌드로 진행 예정
