## 🌙 v0.18.0 — 메뉴부터 결과까지, 손에 잘 잡히는 화면

### 🎯 핵심 변경

상용화 분석 보고서의 **Phase 1: 제품감 정리** 1차 적용. 새로운 기능을 늘리는 대신, 현재 있는 화면들이 상용 게임처럼 보이도록 정보 위계와 모바일 가독성을 한 번에 다듬었습니다. 메인 메뉴, 인게임 HUD, 레벨업 카드, 결과 화면 네 곳을 일관된 톤으로 재배치했습니다.

### 🧭 메인 메뉴 — 1초 안에 무엇을 누를지 보이는 화면

- `PLAY` 버튼을 시각적 단일 1차 액션으로 강조
- 1차 행 — 캐릭터 / 스테이지 / 난이도 (가로 3분할)
- 2차 행 — 상점 / 용어집 / 순위표 (가로 3분할, 더 작은 글자)
- 푸터 — 언어 토글 / 크레딧 (가장 작은 글자)
- 골드 카드에 **"다음 강화까지 N 골드 / ★ 강화 가능"** 힌트 표시 — 메뉴에서 바로 다음 목표가 보임

### 🎮 인게임 HUD — 화면을 덜 가리고, 시간을 더 크게

- 상단 점유 영역 **168 → 112 픽셀** (-33%)
- 생존 시간(`m:ss`)을 상단 중앙에 가장 크게 배치
- HP / XP 바 두께·폰트 크기를 한 손 플레이 기준으로 조정
- 레벨·처치·골드는 시간 아래 한 줄 압축

### 🃏 레벨업 카드 — "DMG +25%" 대신 진짜 변하는 수치

- 헤더에 **신규 / 강화 Lv.X / ★ 진화** 태그
- 현재 → 다음 레벨 표기 (`Lv.3 → Lv.4`)
- 무기별 실제 DMG·CD 변화 수치 — `DMG 18 → 22 · CD 1.06s → 0.93s`
- 패시브는 현재/최대 레벨 표기 (`Lv.2 → Lv.3 (최대 5)`)
- 역할 칩 — `추적 / 근접 / 광역 / 화력 / 생존 / 기동 / 수집 / 편의 / 진화`

### 🏆 결과 화면 — 보상감과 다음 목표를 동시에

- 승리·패배에 따라 **배경 톤이 다르게** (승리는 따뜻한 호박, 패배는 차가운 검적)
- 타이틀에 짧은 스케일 팝 애니메이션
- 획득 골드 **0 → N 카운트업** 트윈
- **"다음 강화까지 N 골드 · ★ 강화 가능"** 라인 — 결과 화면에서 메타 진행이 보임
- 신규 업적은 별도 줄로 강조 (이름 + 보상 골드)

### 🌐 다국어

- KO/EN 신규 키 약 25개 — 메뉴, 레벨업 태그·수치 포맷, 역할 칩, 결과 화면 힌트
- 한국어 표기는 카드형 UI에 맞게 짧게 조정 (`신규`, `강화 · Lv.3`, `★ 진화`)
