### 🛠️ AI 자산 후처리 자동화

ChatGPT(GPT-4o) 출력의 공통 결함을 일괄 보정. `docs/ASSETS_TO_GENERATE.md` §1.1 / §10에 워크플로우 기록.

- **흰 가장자리 자동 crop** — 4면에서 95%+ 흰색 행/열 detect 후 잘라냄
- **target ratio alpha padding** — 비율 보존하며 정확한 native 크기로 nearest 다운샘플
- **외곽 흰 영역 → 알파 (flood-fill)** — 외곽 alpha 0에서 시작해 인접 흰 픽셀만 알파 변환, 가운데 디자인 흰 영역(두루마리 paper, 망루 highlight 등)은 보존
- **1 px navy outline** — 어두운 석판 배경에서 아이콘 가시성 확보
- 패널 / 배경 / 로고 / 아이콘 31장 모두 같은 파이프라인 통과
