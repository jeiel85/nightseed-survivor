## 📚 v0.22.0 — 폰트 가독성 + HUD 아이콘 + 스토리 메뉴

### 🎯 핵심 변경

폰에서 v0.21.0 돌려본 뒤 받은 두 번째 피드백 라운드:

1. *"글씨가 너무 작아서 안보여"* — 첫 외부 리뷰의 "UI overhaul"이 사실은 **폰트 사이즈** 얘기였음
2. *"HUD에 글씨만 있으니 밋밋해"* — 다른 뱀서라이크 류처럼 아이콘/배경이 필요
3. *"정령의 구 아이콘이 경험치 물약과 똑같이 생겼다"* — 사실 모든 무기 아이콘이 같은 Kenney 베이스 스프라이트(작은 물약)의 색 변종이라 구분 안 됨
4. *"스토리 한번에 볼 수 있는 메뉴"* — 해금된 이야기는 다시 읽고, 잠긴 건 "커밍순"

게임플레이는 손대지 않음.

### 🛡 HUD 아이콘 + 경계선

HUD 상단 스탯이 텍스트만 있어서 밋밋했던 부분 보강:

- **Lv** 라벨 앞에 `pickup_xp` 아이콘 (청색 tint)
- **Kills** 라벨 앞에 `icon_moon_dagger` (주황 tint)
- **Gold** 라벨 앞에 `pickup_gold` 아이콘
- 26×26 텍스처, 6px gap, 셀 단위로 묶음
- top bar 하단에 2px 슬레이트 경계선 추가 — 게임 영역과 시각 분리
- HP 라벨에 검은 외곽선 4px — 어떤 fill 색 위에서도 읽힘

### ⚔ 레벨업 카드 아이콘 구분

무기 5종(Moon Dagger / Spirit Orb / Fire Wisp / Star Needle / Thorn Ring) 전부
**Kenney Tiny Dungeon의 작은 물약 스프라이트**를 색만 바꿔서 쓰고 있습니다. XP 픽업도
같은 베이스라 시각적으로 구분이 안 됐어요.

근본 해결은 새 스프라이트 셋이 필요한 큰 작업이라, v0.22에서는 **아이콘 크기 키우고
강한 무기 컬러 tint**로 우회:

- 카드 안 아이콘 70×70 → 96×96px (44% 확대)
- 런타임에 `self_modulate = color.lightened(0.25)` — 무기마다 다른 컬러로 강하게 채색
- 카드 자체의 무기 컬러 보더와 합쳐져서 "장식된 = 무기, 맨몸 = 픽업" 구분 성립

### 📖 스토리 메뉴

메인 메뉴의 **CODEX 버튼이 STORY로 라벨 변경 + 라우트 변경**. 스토리 화면 안에는
우상단 `용어집 →` 버튼이 있어 기존 CodexUI(용어집)로 이어집니다.

스토리 화면 동작:
- `Stages.stages` 순회 (5개 스테이지)
- 해금된 스테이지(`GameData.is_stage_unlocked`): intro / boss_intro / clear 대사 전부 표시. 각 섹션마다 `— 시작 —` / `— 보스 등장 —` / `— 클리어 —` 헤더
- 미해금 스테이지: 🔒 + "이 스테이지를 해금하면 이야기가 열립니다."
- 화자 이름까지 표시 (방랑자, 숲의 파수꾼 등)

Localization 키 9개 신규 추가 (KO/EN).
