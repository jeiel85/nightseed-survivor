## 🌙 v0.14.0 — 조이스틱·카드·사운드 폴리시

게임을 더 손에 잡히게 만드는 세 가지 개선: 화면 어디서나 시작되는 조이스틱, 한눈에 구분되는 레벨업 카드, 박진감 있는 사운드.

### 🕹️ 화면 어디서나 조이스틱

- 기존: 화면 **왼쪽 절반**을 눌러야만 조이스틱 활성화 → 좁고 어색
- 변경: **화면 어디든** 손을 대면 그 자리에 가상 조이스틱 생성
- 멀티터치/다른 UI(레벨업 카드, 결과창)와의 충돌은 기존 일시정지 가드로 그대로 차단

### 🎴 레벨업 카드 — 진짜로 "고르는" UI

- 3장의 카드가 **각자 다른 색 테두리·그림자·배경 틴트**로 한눈에 구분됨 (StyleBoxFlat 적용)
- **카드 어디를 탭해도 선택**됨 — 더 이상 작은 SELECT 버튼을 정확히 누를 필요 없음
- 선택 버튼은 컬러 칩으로 리디자인되어 시각 힌트(▶ 선택) 역할
- 타이틀 색상이 옵션 색을 따라가 식별성 ↑
- 카드 간 간격 20 → 28px

### 🥁 사운드 박진감 (BGM에 드럼 추가)

- **BGM 드럼 라인**:
  - `game` 무드: 4분 킥 + 8분 하이햇 (그루브)
  - `boss` 무드: 더 강한 킥 + 16분 하이햇 (텐션)
  - `menu`는 그대로(잔잔)
- **새 SFX 헬퍼**:
  - `punch_tone` — 어택에 노이즈 트랜션트 + 하모닉. **kill/hit**에 적용 → "팡"
  - `boom` — 저음 스윕 + 노이즈 테일. **boss_appear** 교체 → 묵직한 등장음
  - `arpeggio`에 harmonic/sub 옥타브 파라미터 추가 → **level_up / evolve / victory / pickup_gold**가 풍부해짐
- **damage thud**를 약간 길고 무겁게 (0.18→0.22s, 0.45→0.55 amp)
