## 🌙 v0.16.0 — 밤의 씨앗이 말을 걸기 시작했다

스토리를 게임 안으로 들여왔습니다. 스테이지 입장 / 보스 등장 / 클리어 시점에 비차단 자막이 흘러가고, 메인 메뉴에서 용어집(CODEX)을 열어 세계관을 따로 읽을 수 있습니다. 자동 검증 단계도 CI에 추가되어 앞으로 런타임 회귀를 빌드 전에 잡습니다.

### 📜 인게임 스토리 자막

- 스테이지 시작 시 **방랑자의 한 줄**이 상단에 페이드 인 — 스테이지가 던지는 첫 단서
- 보스 등장 시 **"봉인의 파수꾼이 깨어납니다"** 경고 → 파수꾼과 방랑자의 짧은 대화
- 승리 시 **클리어 대사** + Result 화면에 "기억의 조각을 되찾았습니다" 부제
- 자막은 게임을 멈추지 않습니다 — 보스가 곧장 따라옵니다

### 📖 용어집 (CODEX)

- 메인 메뉴에 **CODEX** 진입 추가
- Nightseed / Moonlight / Vagrant's Blade 3개 용어 수록
- 한국어/영어 토글 동기화
- 향후 도감/잠금-해금 확장을 위한 데이터 기반 구조 (`story_terms.json`)

### 🛡️ CI 자동 검증 추가

- 빌드 전에 **헤드리스 스모크 단계** 신규: 메인 메뉴 + GameRoot 씬을 헤드리스로 짧게 실행해 `SCRIPT ERROR` / `push_error` / `Invalid get index` 같은 런타임 회귀를 잡고 실패시킴
- 이번 푸시부터 깨진 시그널 연결·NodePath 오타·JSON 파싱 실패가 빌드 시간 들이기 전에 드러납니다

### 🧹 리포지토리 위생

- Godot 엔진 바이너리는 더 이상 프로젝트 루트에 두지 않음 (사용자 개인 PATH로 이동, `.gitignore`는 이미 보호)
- `godot/android/build/assetPacks/*/build/` `src/` 그래들 산출물 `.gitignore`에 추가
