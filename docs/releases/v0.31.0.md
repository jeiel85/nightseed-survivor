## 🌙 v0.31.0 — 뒤로 가기 정비 + 설정 화면

### 🎯 핵심 변경

뱀서라이크 모바일 게임이 갖춰야 할 기본 UX 컨벤션을 한 번에 점검했습니다 (이슈 [#3](https://github.com/jeiel85/nightseed-survivor/issues/3)). 메인 메뉴에 **BGM/SFX 음량과 진동 토글이 있는 설정 화면**이 새로 생겼고, 종료 다이얼로그는 게임 톤의 커스텀 모달로 교체했습니다. 일시정지 메뉴에서는 이제 배경음도 함께 정지하고, 폰을 잠갔다 풀어도 게임이 자동으로 일시정지 메뉴에 머무릅니다.

---

### 🆕 추가 — 설정 화면

메인 메뉴 우하단의 **"설정"** 버튼으로 진입.

- **BGM 슬라이더** (0~100%, 5% 단위) — 움직임 즉시 반영
- **SFX 슬라이더** — 효과음 음량도 같은 방식
- **진동 토글** — 단말이 햅틱을 지원할 때만, 토글 ON 순간 짧은 진동으로 확인 피드백
- 모든 변경은 `GameData`에 즉시 저장 — 다시 켜도 유지됩니다

---

### 🛠 다듬기 — 뒤로 가기

- **종료 다이얼로그 게임 톤 교체** — 기존 Godot 기본 `AcceptDialog` (OS 위젯 톤) → `PanelContainer + ButtonStyles` 기반 커스텀 모달. "취소"(좌측) / "종료"(우측, 빨강) 두 버튼.
- **다이얼로그 떠 있을 때 Back → 다이얼로그만 닫기** — 이전엔 가드 로직 때문에 메뉴에 갇히는 케이스가 있었습니다.
- **레벨업 카드 표시 중 Back 무시 명시화** — 동작은 v0.29.0부터 그대로지만 의도를 `LevelUpUI._notification` 에 코드로 남겨 향후 재사용 시 안전.

---

### 🛠 다듬기 — 일시정지 / 앱 복귀

- **일시정지 메뉴에서 BGM/SFX 정지** — `AudioManager.set_paused(true)` 추가. BGM 플레이어 + 8개 SFX 풀이 모두 같은 지점에서 멈췄다가 재개 시 이어집니다.
- **앱 복귀 시 자동 일시정지 메뉴** — 폰을 잠갔다가 풀거나 다른 앱에 갔다 돌아오면 (`NOTIFICATION_APPLICATION_RESUMED`) 의도치 않은 사이 적이 다가오지 않도록 자동으로 일시정지 메뉴를 띄웁니다. v0.29.0의 자동 저장과 짝을 이루는 안전 장치.

---

### 🛠 다듬기 — 터치 영역

- **캐릭터/스테이지 선택 버튼 높이 70 → 88px** (≈47dp → ≈59dp) — Material 권장 최소 48dp 보장. 작은 폰에서 손가락 끝으로 정확히 누르기 쉬워집니다.
