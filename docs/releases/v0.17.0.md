## 🌙 v0.17.0 — 한 손으로 5분, 더 잘 보이는 전장

### 🎯 핵심 변경

스테이지 길이를 **7분 → 5분**으로 압축. 짧아진 만큼 난이도 곡선을 가파르게 끌어올려, 막판 핵 앤 슬래시 절정은 그대로 유지합니다. 동시에 카메라를 1.5배 줌인하고 픽업 시각 구분을 강화해 "캐릭터가 너무 작다 / 아이템과 적 구분이 안 된다"는 모바일 피드백에 답했습니다. 펀치홀·노치 단말의 시스템 시계와 HUD가 겹치던 문제도 해결됐습니다.

### ⏱ 5분 페이스로 압축

- 모든 스테이지 **총 5분**, 보스 4분 30초 등장 — Cursed Tomb는 난이도 차이 보존을 위해 5분 30초 / 5분 보스
- 미니보스 등장 60s / 120s / 180s / 240s (기존 80·160·240·320 → 압축)
- 난이도 곡선 압축: 5분차에 HP ~2.4×, 데미지 ~1.7×, 속도 ~1.28× → 기존 7분 막판 강도 그대로
- 5개 스테이지의 wave 데이터 시간 축 비례 재배치

### 🔎 더 큰 화면, 더 잘 보이는 픽업

- 카메라 줌 **1.5×** — 캐릭터·몬스터·아이템 모두 1.5배 커 보임
- XP 보석: **시안 글로우 + 상하 바운싱** + 알파 펄스
- 골드 동전: **황금 글로우 + 회전 애니메이션** + 알파 펄스
- 픽업 스프라이트 1.4× → 1.6× — 적과 한눈에 구분
- 적 스폰 반경을 좁힌 시야에 맞춰 재조정 (750 → 520)

### 📱 펀치홀·노치 단말 대응

- HUD가 `DisplayServer.get_display_safe_area()` 를 읽어 컷아웃만큼 안쪽으로 자동 패딩
- `AndroidManifest`: `windowLayoutInDisplayCutoutMode="shortEdges"` 추가
- 펀치홀 단말에서 시스템 시계와 인게임 시간/HP가 겹치지 않음
- 펀치홀 없는 단말은 안전영역 = 화면 전체 → 영향 없음
- 화면 회전·창 크기 변경 시 자동 재계산
