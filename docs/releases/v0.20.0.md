## ✨ v0.20.0 — UI 디테일 다듬기

### 🎯 핵심 변경

비공개 테스트에서 받은 첫 외부 리뷰(*"게임 진짜 좋고 중독성 있는데 UI는 좀 손봤으면"*)
에 대응한 폴리시 라운드. 게임플레이는 그대로 두고, 사용자가 가장 자주 보는 세 화면
— **메인 메뉴 / 인게임 HUD / 레벨업 카드** — 의 디테일만 짧게 손봤습니다.

---

### 📱 메인 메뉴 — 푸터 정리

`Language` / `Credits` 두 버튼이 메인 영역에서 빠지고 **상단 우측 코너의 작은 행**으로
이동했습니다. 첫 화면이 PLAY + 6개 액션 버튼 + 골드 카드로 정리되어 시각적 밀도가
줄어듭니다.

- Language 버튼 라벨도 `Language: English` → `English` 한 줄로 단축
- 메인 영역 행 수: 1차 + 2차 + 푸터 = 3행 → 1차 + 2차 = 2행

---

### ❤ 인게임 HUD — HP 바 동적 색상

HP 바가 잔여 비율에 따라 색이 바뀝니다 — "잘못 죽었나" 감을 줄이려는 1차 가독성 변화.

| 잔여 HP | 색 |
|---|---|
| 55% 초과 | 초록 |
| 30 ~ 55% | 호박 → 빨강 (보간) |
| 30% 이하 | 빨강 + 알파 펄스 (0.55 ↔ 1.0, 0.35s 사인 루프) |

기본 ProgressBar의 fill/background 스타일박스를 직접 오버라이드해 둥근 모서리도 같이
들어갔습니다.

---

### 🃏 레벨업 카드 — 탭 피드백

카드 전체가 탭 영역인 건 기존부터 동작했지만 **눌렀을 때 시각 반응이 없어서** 카드가
"진짜 탭되는지" 의심스러운 순간이 있었습니다. 이제 누르는 순간 카드가 살짝 작아졌다가
(`scale 0.96`, 0.08s, 사인 ease-out) 떼면 1.0으로 복귀하면서 선택 처리됩니다. 터치 ·
마우스 양쪽 적용.
